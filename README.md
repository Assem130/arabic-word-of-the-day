## الفكرة

لا تعرض **كَلِمات** تعريفًا سريعًا فحسب؛ بل تمنح كل لفظة مساحة تكفي لتأملها واستعمالها. تظهر كلمة عربية واحدة ثابتة خلال اليوم، مكتملةً بضبطها ومعناها ومثالها وخصائصها الصرفية، ثم تنتقل إلى سجل محلي يمكنك الرجوع إليه.

المشروع مبني بواجهات الويب الأساسية، من دون إطار عمل أو خطوة بناء.

## ما الذي تقدمه التجربة؟

| الميزة | التفاصيل |
|---|---|
| كلمة اليوم | اختيار يومي ثابت لا يتغير عند تحديث الصفحة |
| سياق لغوي متكامل | المعنى، والضبط، والوزن، والجذر، والتصنيف، ومثال الاستخدام |
| النطق | قراءة صوتية عبر واجهة النطق المدمجة في المتصفح |
| المخزون اللغوي | سجل للكلمات السابقة محفوظ محليًا في المتصفح |
| المشاركة | مشاركة تفاصيل الكلمة أو نسخها مباشرة |
| الموعد التالي | عدّاد تنازلي حتى ظهور كلمة اليوم التالية |
| تجربة عربية | واجهة متجاوبة من اليمين إلى اليسار، مصممة حول الحرف العربي |

## الرحلة

1. تبدأ من واجهة تحريرية تحتفي باللغة العربية.
2. تنتقل إلى كلمة اليوم وتستكشف بنيتها ومعناها وسياقها.
3. تُحفظ الكلمة تلقائيًا في مخزونك اللغوي داخل المتصفح.
4. تعود لاحقًا إلى الكلمات التي مررت بها، من دون حساب أو خادم بيانات.

## التشغيل المحلي

يتطلب المشروع **Python 3** فقط لتشغيل خادم محلي يقدّم الملفات بترميز UTF-8:

```powershell
git clone https://github.com/Assem130/arabic-word-of-the-day.git
cd arabic-word-of-the-day
python server.py
```

ثم افتح:

```text
http://localhost:8000
```

يمكن فتح `index.html` مباشرة أيضًا، لكن الخادم المحلي هو المسار الموصى به لضمان عرض العربية بصورة صحيحة.

## بنية المشروع

```text
arabic-word-of-the-day/
├── index.html      # الواجهة الافتتاحية
├── word.html       # صفحة كلمة اليوم
├── revamp.css      # الهوية البصرية والتجاوب
├── revamp.js       # حركة الواجهة الافتتاحية
├── app.js          # الكلمات ومنطق التجربة
└── server.py       # خادم محلي بسيط بترميز UTF-8
```

## الخصوصية

لا يتطلب المشروع تسجيل الدخول، ولا يرسل سجل تعلّمك إلى أي خادم. تُحفظ الكلمات التي شاهدتها في `localStorage` داخل متصفحك فقط، ويمكن حذفها بمسح بيانات الموقع.

## التقنية

- HTML دلالي وواجهة RTL
- CSS متجاوب وهوية بصرية عربية
- JavaScript أصلي لإدارة كلمة اليوم والحالة المحلية
- Web Speech API للنطق
- GSAP وScrollTrigger لحركة الواجهة الافتتاحية
- Google Fonts لخطّي Amiri وOutfit
